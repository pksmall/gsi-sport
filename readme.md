# GSI Sport at Laravel 5.6

This working code can be viewed at [GSI Test Site](https://gsi.dtgb.solutions/products) 

## Install

#### git clone
```
$ git clone https://github.com/pksmall/gsi-sport
```
#### copy and edit .env file
```
$ cp .env.example .env
$ vi .env
```

#### install vendors
```
$ php composer.phar install
```

#### create database
```
$ php artisan migrate
```
